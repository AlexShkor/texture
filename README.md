# Substance Scientist

# Install

Clone the repository.

```bash
$ git clone https://github.com/substance/scientist.git
```

Navigate to the source directory.

```bash
$ cd scientist
```

Install via npm

```bash
$ npm install
```

Start the dev server

```bash
$ npm run start
```

And navigate to [http://localhost:5001](http://localhost:5001)
